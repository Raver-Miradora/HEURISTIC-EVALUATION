Computer Management System

This heuristic evaluation examines the usability of the Computer Laboratory Management System using Jakob Nielsen's 10 usability principles. The assessment identifies critical usability issues across key functionalities. By analyzing the system's alignment with real-world user expectations, this report highlights opportunities to streamline navigation and enhance user control.
